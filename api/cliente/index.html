Documentação completa para integração

Token de Autenticação

Para testar os endpoints autenticados, use o token abaixo no header:

Authorization: Bearer 781e5e245d69b566979b86e28d23f2c7

Endpoints Públicos (Sem Autenticação)

GET Listar Tipos de Serviço

URL: /api/cliente/ClienteApi.php/tipos-servico

Descrição: Retorna todos os tipos de serviço disponíveis.

Exemplo de Request:
GET https://chamaservico.tds104-senac.online/api/cliente/ClienteApi.php/tipos-servico
Exemplo de Response:
{ "sucesso": true, "tipos_servico": [ { "id": 1, "nome": "Eletricista", "descricao": "Serviços elétricos residenciais" }, { "id": 2, "nome": "Encanador", "descricao": "Reparos hidráulicos" } ] }

POST Login de Cliente

URL: /api/cliente/ClienteApi.php/login

Descrição: Autentica um cliente e retorna seus dados + token.

Body (JSON):
{ "email": "cliente@example.com", "senha": "senha123" }
Exemplo de Response:
{ "sucesso": true, "usuario": { "id": 1, "nome": "João Silva", "email": "cliente@example.com", "tipo": "cliente" }, "token": "781e5e245d69b566979b86e28d23f2c7" }

POST Registrar Novo Cliente

URL: /api/cliente/ClienteApi.php/registro

Descrição: Cria uma nova conta de cliente.

Body (JSON):
{ "nome": "Maria Santos", "email": "maria@example.com", "senha": "senha123", "telefone": "(11) 99999-9999" }
Exemplo de Response:
{ "sucesso": true, "cliente_id": 5, "mensagem": "Cliente cadastrado com sucesso" }

Endpoints Autenticados

Requerem o header Authorization: Bearer [TOKEN]

GET Buscar Perfil do Cliente

URL: /api/cliente/ClienteApi.php/perfil?cliente_id={id}

Headers:
Authorization: Bearer 781e5e245d69b566979b86e28d23f2c7
Exemplo de Response:
{ "sucesso": true, "perfil": { "id": 1, "nome": "João Silva", "email": "cliente@example.com", "telefone": "(11) 98765-4321" }, "enderecos": [...] }

GET Listar Solicitações

URL: /api/cliente/ClienteApi.php/solicitacoes?cliente_id={id}

Parâmetros opcionais:
  • status - Filtrar por status
  • busca - Buscar por título/descrição

POST Criar Nova Solicitação

URL: /api/cliente/ClienteApi.php/solicitacoes

Body (JSON):
{ "cliente_id": 1, "titulo": "Reparo elétrico urgente", "descricao": "Tomada com problema", "tipo_servico_id": 1, "endereco_id": 1, "urgencia": "alta", "orcamento_estimado": 200.00 }

GET Listar Propostas Recebidas

URL: /api/cliente/ClienteApi.php/propostas?cliente_id={id}

Parâmetros opcionais:
  • status - Filtrar por status (pendente, aceita, recusada)
  • solicitacao_id - Propostas de uma solicitação específica

POST Aceitar Proposta

URL: /api/cliente/ClienteApi.php/aceitar-proposta

Body (JSON):
{ "cliente_id": 1, "proposta_id": 5 }

POST Adicionar Endereço

URL: /api/cliente/ClienteApi.php/enderecos

Body (JSON):
{ "cliente_id": 1, "cep": "01310-100", "logradouro": "Av. Paulista", "numero": "1000", "complemento": "Apto 101", "bairro": "Bela Vista", "cidade": "São Paulo", "estado": "SP", "principal": 1 }

DELETE Excluir Endereço

URL: /api/cliente/ClienteApi.php/enderecos

Body (JSON):
{ "cliente_id": 1, "endereco_id": 3 }

Códigos de Status HTTP

Código Significado
200 Sucesso
401 Não autorizado (token inválido)
404 Endpoint não encontrado
405 Método não permitido
500 Erro interno do servidor
